
### 2️⃣ RODAR (30 segundos)

## 💡 DICAS IMPORTANTES

### 🔥 Hot Reload
Você edita → Salva (Ctrl+S) → Página atualiza automaticamente
Isso torna o desenvolvimento **extremamente rápido**!

### 🎨 Tailwind CSS
- Use classes predefinidas: `bg-red-500`, `text-blue-600`
- Cores vão de 50 até 950 (mais escuro)
- Exemplo: `bg-blue-500` (médio), `bg-blue-900` (escuro)

### 🚀 Atualizar Navegador
Se mudanças não aparecem: **Ctrl + Shift + R** (força atualizar)

### 📱 Testar Responsividade
Pressione **F12** no navegador → Clique ícone celular → Escolha dispositivo

### 🐛 Debugar Erros
Pressione **F12** → Aba **Console** → Veja erros em vermelho
